# Загрузка иконок с helldivers.wiki.gg

`data.ts` уже содержит поле `imageUrl` для всех 234 предметов вида:

```
/assets/weapons/primary/assault-rifles/liberator.png
```

Файлов по этим путям пока физически нет — их нужно скачать этим скриптом
(среда, в которой Claude собирал проект, не имеет доступа к
helldivers.wiki.gg, поэтому скачать за вас напрямую не получилось).

## Запуск (на VPS или локально)

```bash
cd scripts
python3 -m venv venv && source venv/bin/activate   # опционально
pip install -r requirements.txt
python3 download_images.py
```

Скрипт:
- пройдётся по всем 234 предметам из `items.json`;
- для каждого попробует найти иконку на вики (сначала по точному названию
  страницы, затем через поиск, для брони/шлемов — через список файлов на
  странице сета);
- сохранит найденные иконки как PNG прямо в нужные подпапки `assets/`;
- пропустит уже скачанные файлы (запуск можно прерывать и продолжать);
- в конце выведет и сохранит в `report.json` список предметов, для которых
  иконку найти не удалось — их придётся добавить руками (см. ниже).

Полезные флаги:

```bash
python3 download_images.py --force              # перекачать всё заново
python3 download_images.py --only liberator ce-27-ground   # только выбранные id
python3 download_images.py --delay 1.0           # вежливее к серверу вики
```

## Если что-то не нашлось

Из-за разницы в названиях (например, у брони на вики одна страница на
весь сет с бронёй/шлемом/плащом сразу) часть предметов скрипт может не
найти автоматически. Для них:

1. Откройте `report.json`, найдите `id` и `nameEn`.
2. Найдите страницу предмета на https://helldivers.wiki.gg/wiki/{nameEn}.
3. Скачайте иконку вручную и положите в путь, указанный в `imageUrl` этого
   предмета в `data.ts` (или измените сам `imageUrl`, если удобнее).

## После скачивания

`vite.config.ts` в этом проекте уже настроен раздавать файлы из `assets/`
по пути `/assets/...` — ничего дополнительно прописывать не нужно, просто
запустите `npm run dev` / `bun run dev` и картинки подхватятся.
